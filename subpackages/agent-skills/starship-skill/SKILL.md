# starship-tracker：星舰进度追踪

本 SKILL 提供 SpaceX 星舰（Starship）计划的查询能力：下一飞组合体状态、最新动态、下次试飞时间、星舰基地道路封闭。数据来自小程序云端同步（NSF/LL2/官方推文，已翻译中文），时间均为北京时间。

## 意图分流

- 用户问「星舰最新进展/造到哪了/B16 或 S37 状态」→ getStarshipStatus
- 用户问「星舰最近有什么新闻/动态」→ getStarshipUpdates
- 用户问「星舰下次试飞什么时候/第 X 次试飞时间」→ getStarshipNextFlight
- 用户问「星舰基地封路了吗/最近有测试吗」→ getRoadClosures
- 用户笼统问「星舰怎么样了」→ 先调 getStarshipStatus 展示状态卡片，再按用户追问分流

## 业务流程

1. getStarshipStatus 返回后展示状态卡片并概括，等待用户追问，不要主动连环调用其他接口。
2. 用户追问试飞时间 → getStarshipNextFlight → 介绍任务并提示可进小程序看倒计时与直播；若未排期则如实告知，不编造日期。
3. getStarshipUpdates 返回多条动态时，挑最重要的 2-3 条概括，不逐条罗列。
4. getRoadClosures 返回封路时可补充说明「封路通常预示测试或试飞临近」。

## 跨接口约束

1. 所有接口无必填参数，禁止编造参数；接口失败或无数据时如实告知，禁止以相同参数立即重试。
2. 试飞时间可能推迟，回答时间时附「以实际为准」。
3. 时间字段均为北京时间，直接展示，不要再换算时区。
4. 深度内容（建设进度图、硬件清单、基地地图、动态视频）引导用户进入小程序星舰进度页。
