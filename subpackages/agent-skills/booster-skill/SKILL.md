# booster-tracker：可回收火箭助推器查询

本 SKILL 提供全球可回收火箭助推器的查询能力：单枚助推器战绩（按编号）与回收统计总览。数据来自小程序云端同步的助推器族谱（SpaceX 猎鹰系列 + 全球其他可回收火箭）。

## 意图分流

- 用户问「B1067 飞了多少次」「B1080 状态」等带编号的问题 → getBoosterInfo（serial 取用户原文）
- 用户问「回收成功率」「哪枚助推器复用最多」「火箭回收统计」→ getRecoveryStats
- 用户想查某枚助推器但没给编号 → 先调 getRecoveryStats 展示排行让用户挑选，不要编造编号

## 业务流程

1. getRecoveryStats 返回后概括统计并列出排行，等待用户挑选，不要主动逐枚调用 getBoosterInfo。
2. 用户点名某枚助推器 → getBoosterInfo → 介绍战绩与最近任务，并提示可进小程序看完整飞行史。

## 跨接口约束

1. serial 一律取用户提到的编号原文或 getRecoveryStats 返回的 serial 原值，禁止编造。
2. 接口失败或未找到时如实告知，禁止以相同参数立即重试，禁止编造飞行次数与着陆数据。
3. 完整族谱、飞行史时间线、着陆方式明细引导用户进入小程序「全球可回收火箭族谱」页面。
